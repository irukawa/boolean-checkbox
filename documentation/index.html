Introdução

boolean-checkbox é o componente que inclui na página apenas um checkbox.

Como utilizá-lo?

Antes de começar, é necessário instalar as seguites ferramentas:

  • Node.js (versão mais recente).
  • Bower (versão mais recente).

Caso já possua as ferramentas instaladas, basta seguir os passos abaixo para começar a desenvolver:

  1. No terminal acesse a pasta raiz do projeto.
  2. Execute o comando: bower install --save boolean-checkbox.
  3. Adicione o módulo do componente no app.js da aplicação: 
    angular.module('app', [
    'booleanCheckbox'
]);
  4. Ao final do processamento o bower terá baixado o componente para dentro do projeto.

Todos os arquivos necessários para o funcionamento do componente podem ser encontrados na pasta PROJETO/bower_components/boolean-checkbox.

Parâmetros

Os parâmetros abaixo com o tipo em vermelho são obrigatórios.

Parâmetro Tipo Valor padrão Detalhamento
np-id string - Esse atributo refere-se ao identificador do componente.
np-value string - Valor que será exibido para seleção.
np-selected-value boolean - Valor que será retornado para o modelo.
np-object-value object - Permite retornar um objeto para o atributo np-selected-value. Se não informado o valor retornado será true ou false.
np-optional boolean true Atributo booleando que diz se o campo será opcional.
np-label string - Título que fica em cima do campo.
np-size string 12 Atributo que diz o tamanho do campo, podendo ser de 0 a 12.
np-disabled boolean false Diz se o campo ficará desabilitado para edição.
np-required-message string Campo obrigatório. Diz qual mensagem será a ser exibida, caso o campo seja de preenchimento obrigatório.
np-on-change function - Função que será disparada no evento onchange.
np-on-click function - Função que será disparada no evento onclick.
np-accessibility-message number - Esse atributo tem a finalidade de ajudar na acessibilidade dos usuários. Quando informado a mensagem ficará escondida na tela, porém o leitor de tela irá falar a mensagem toda vez que o campo tiver o foco. Essa mensagem deve ser sempre bem explicativa.
np-validator-message string - Define a mensagem de validação que será exibida caso a função ou variável passada na diretiva validator invalide o campo.

Exemplo

Abaixo um exemplo da utilização do componente.

Pagina HTML 
<boolean-checkbox
  np-id="booleanCheck"
  np-label="Selecione uma opcao abaixo:"
  np-value="value"
  np-selected-value="user"
  np-optional="false" validator="validar()"
  np-validator-message="teste"
  np-on-click="click()">
</boolean-checkbox>
Controller.js 
$scope.value = "Português"
$scope.submit = function() {
    $scope.submitted = true;
    if ($scope.form.$valid) {
        alert("válido");
    } else {
        alert("inválido");
    }
}
$scope.click = function() {
     console.log("Selecionado: " + $scope.user);
}
